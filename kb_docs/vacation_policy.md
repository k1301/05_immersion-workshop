# 휴가 정책 가이드

## 1. 연차 휴가

### 1.1 연차 발생 기준
- 입사 1년 미만: 월 1개 발생 (최대 11개)
- 입사 1년 이상: 연 15개 발생
- 3년 이상 근속: 2년마다 1개 추가 (최대 25개)

### 1.2 연차 사용 방법
1. 사내 포털(http://portal.company.com)에서 휴가 신청
2. 팀장 승인 필요
3. 최소 3일 전 신청 권장 (긴급 시 당일 가능)

### 1.3 연차 이월
- 미사용 연차는 다음 해 이월 가능 (최대 11개)
- 이월된 연차는 해당 연도 내 사용 필수

## 2. 병가

### 2.1 유급 병가
- 연 10일까지 유급 병가 제공
- 진단서 제출 시 추가 병가 가능

### 2.2 신청 방법
1. IT 헬프데스크에 병가 신청 티켓 생성
2. 진단서는 복귀 후 3일 이내 제출

## 3. 경조사 휴가

### 3.1 경조사 종류 및 일수
- 본인 결혼: 5일
- 직계 가족 경조사: 3일
- 형제자매 경조사: 1일

### 3.2 신청 방법
- 경조사 증빙 자료와 함께 사전 신청
- 긴급 시 사후 신청 가능

## 4. 리프레시 휴가

### 4.1 대상
- 3년 이상 근속 직원

### 4.2 내용
- 연 5일 추가 유급 휴가
- 연차와 별도 제공
- 최소 3일 연속 사용 권장
