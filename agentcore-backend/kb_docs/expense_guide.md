# 경비 처리 가이드

## 1. 경비 신청 대상

### 1.1 업무 관련 경비
- 출장비 (교통, 숙박, 식비)
- 접대비
- 교육비
- 업무용 소모품
- 도서 구입비

### 1.2 복리후생비
- 팀 회식비
- 명절 선물비
- 경조사비

## 2. 경비 한도

### 2.2 출장비
- 국내 출장
  - 교통비: 실비
  - 숙박비: 1박당 최대 15만원
  - 식비: 1일 3만원
- 해외 출장
  - 교통비: 이코노미 클래스 (임원은 비즈니스 가능)
  - 숙박비: 지역별 상이 (미국 1박 $200)
  - 일비: 지역별 상이 (미국 $100/일)

### 2.3 접대비
- 팀장급 이하: 건당 50만원
- 팀장급: 건당 100만원
- 임원: 건당 200만원

### 2.4 교육비
- 외부 교육: 연 300만원
- 컨퍼런스: 연 2회
- 도서: 월 10만원

## 3. 경비 신청 절차

### 3.1 사전 결재
1. 경비 포털에서 사전 결재 신청
2. 팀장 승인
3. 경비 사용
4. 증빙 자료 업로드

### 3.2 사후 정산
1. 경비 사용 후 7일 이내 정산
2. 영수증 원본 또는 전자 영수증 첨부
3. 팀장 승인
4. 다음 월 급여 시 지급

### 3.3 법인카드 사용
- 법인카드 발급 대상: 팀장급 이상
- 월 사용 한도: 직급별 상이
- 개인 용도 사용 절대 금지

## 4. 증빙 자료

### 4.1 필수 증빙
- 세금계산서 (10만원 이상)
- 신용카드 영수증
- 현금영수증
- 간이영수증 (5만원 미만만 가능)

### 4.2 증빙 예외
- 교통비: 대중교통 카드 내역
- 주차비: 주차 영수증
- 택시비: 택시 영수증 또는 앱 결제 내역

### 4.3 증빙 누락 시
- 경위서 작성 및 팀장 확인
- 3회 이상 누락 시 경고 조치

## 5. 특수 경비

### 5.1 재택근무 지원금
- 월 10만원 (통신비, 전기세)
- 별도 신청 불필요 (자동 지급)

### 5.2 자기계발비
- 연 100만원 (도서, 온라인 강의 등)
- 분기별 신청 가능

### 5.3 건강검진비
- 연 1회 건강검진 지원 (최대 50만원)
- 제휴 병원 이용 시 전액 지원

## 6. 주의사항

### 6.1 금지 사항
- 개인 용도 경비 청구
- 허위 영수증 제출
- 중복 청구
- 타인 명의 카드 사용

### 6.2 위반 시 조치
- 1차: 경고
- 2차: 징계 (감봉)
- 3차: 해고
